Structure for One-Way ANOVA

For one-way ANOVA, we have a single factor-grouping which contains information about the different groups at this factor-level.

Each factor-level is a vector (or column in a data frame) that has a group name for each measurement.

Long form data will have $N$ rows each group will occupy $n_i$ rows for each group.

The easy terms

We can think of Two-Way ANOVA as a nesting of One-Way ANOVAs. That is, we end up with, essentially, three One-Way ANOVAs: one for each factor-level and one for the combined factor levels.

For factor-levels $A$ and $B$: \begin{equation} \begin{split} \left\{ \mathcal{F}_A = \frac{\color{lightgreen}{SS_{A}}}{SS_{within}}, \\ \mathcal{F}_B = \frac{\color{lightblue}{SS_{B}}}{SS_{within}}, \\ \mathcal{F}_{A*B} = \frac{SS_{\color{lightgreen}{A}*\color{lightblue}{B}}}{SS_{within}}\quad \right\} \end{split} \end{equation}

\begin{equation} SS_A = \sum_{i = 1}^{k_A} n_{A_i}\left( \hat{y}_{A_i} - \bar{y}\right)^2,\quad {\scriptstyle \mathrm{dof}=k_A - 1} \end{equation}

\begin{equation} SS_B = \sum_{i = 1}^{k_B} n_{B_i}\left( \hat{y}_{B_i} - \bar{y}\right)^2,\quad {\scriptstyle \mathrm{dof}=k_B - 1} \end{equation}

\begin{equation} SS_{within} = \sum_{i=1}^{k_{AB}}\sum_{j=1}^{n_i}(y_{ij} - \hat{y}_{i})^2,\quad {\scriptstyle \mathrm{dof}=N-k_A\times k_B} \end{equation}

Two-Way ANOVA

The complicated term

The interaction term, $SS_{A*B}$, takes into account the fact that a measurement belongs to both $A$ and $B$ factors, which have some effect, $\epsilon$, that needs to be accounted for.

$\epsilon$ is the sum of each factor-level effects.

Factor-level effects are actually the residuals of the between groups variances.

Like other $SS$ terms, $SS_{A\cdot B}$ is the residual of the true grouping minus the sum of the factor-level effects.

\begin{equation} \epsilon = \epsilon_A + \epsilon_B, \end{equation}

\begin{equation} \epsilon_i = (\hat{y}_i - \bar{y}) \\ \end{equation} $$\therefore \epsilon_{A_i} = (\hat{y}_{\{A\, \mid\, i\, \in A \}} - \bar{y})$$

\begin{equation} SS_{A\cdot B} = \sum_{i=1}^{k_{A\cdot B}}\color{orange}{n_{A\cdot B\: _i}} \left( \color{orange}{(\hat{y}_{A\cdot B\: _i} - \bar{y})} - (\color{lightblue}{\epsilon_{A_i} + \epsilon_{B_i}}) \right)^2 \end{equation}

With substitution, our interaction term becomes:

$$SS_{A\cdot B} = \sum_{i=1}^{k_{A\cdot B}}n_{A\cdot B\: _i} \left( (\hat{y}_{A\cdot B\: _i} - \bar{y}) - ((\hat{y}_{\{A\, \mid\, i\, \in A \}} - \bar{y}) + (\hat{y}_{\{B\, \mid\, i\, \in B \}} - \bar{y})) \right)^2$$

Which simplifies:

$$SS_{A\cdot B} = \sum_{i=1}^{k_{A\cdot B}}n_{A\cdot B\: _i} \left( \hat{y}_{A\cdot B\: _i} - \hat{y}_{\{A\, \mid\, i\, \in A \}} - \hat{y}_{\{B\, \mid\, i\, \in B \}} + \bar{y}) \right)^2$$

Though, to keep it in the same form, we write:

$$SS_{A\cdot B} = \sum_{i=1}^{k_{A\cdot B}}n_{A\cdot B\: _i} \left( \hat{y}_{A\cdot B\: _i} - (\hat{y}_{A_i} + \hat{y}_{B_i} - \bar{y}) \right)^2$$

Two-way ANOVA Interpretation

Inference for the Two-way ANOVA table involves first checking the interaction p-value to see if we can reject the null hypothesis that the additive model is sufficient. If that p-value is smaller than $\alpha$, then the adequacy of the additive model can be rejected and you should conclude that both factors affect the outcome, and the the effect changes in one factor depend on the level of the other factor, i.e. there is an interaction between the explanatory variables.

When the interaction term is significant, the usual advice is that you should not test the effects of the individual groups as this is misleading.

The typical approach is to then break apart the factors and compute One-Way ANOVA on each factor term separately.

$$y_{ijkr} = \bar{y} + var_{A_i} + var_{B_j} + var_{C_k} + var_{AB_{ij}} + var_{AC_{ik}} + var_{BC_{jk}} + var_{ABC_{ijk}} + \epsilon_{ijkr}$$

Post-hoc Testing Problem

In a test at $\alpha = 0.05$, there is a 5% chance of getting a false positive if the null hypothesis is true. In 100 tests, each at $\alpha = 0.05$, we "expect 5 false positives" if all 100 null hypotehses are true, but really, the probability of getting $k$ false-positives is determinable through the binomial distribution, $p(k\, |\, \alpha)$.

We can also determine the probability of getting k false positives by bootstrap. Create a deck with 100 cards, 5 of which are red, then resample with replacement and count the red cards.
